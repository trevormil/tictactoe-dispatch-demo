# CLAUDE.md

Vanilla HTML/CSS/JS tic-tac-toe. No framework, no build step, no dependencies —
keep it that way. `index.html` + `style.css` + `script.js` is the whole app.

- Match the existing plain-DOM style in `script.js`.
- No new dependencies or bundlers; features ship as plain JS/CSS.
- Keep the app fully playable by opening `index.html` directly (file://).
