# tictactoe-dispatch-demo

A deliberately minimal vanilla-JS tic-tac-toe web app, used as a sandbox for
testing [Dispatch](https://github.com/jwolberg/dispatch) — the spec-chat →
issue → `claude-code-action` build → PR → board pipeline.

Open `index.html` in a browser to play. No build step, no dependencies.

Intentionally missing features (good first Dispatch tickets):

- Highlight the winning line
- Score tracking across games
- A simple computer opponent
- Sound effects / animations
